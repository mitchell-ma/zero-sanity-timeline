# zero-sanity-timeline
Arknights: Endfield timeline and rotation calculator

React 19+. Typescript.

Project structure:
- Views/Components: src/view
- Pages: src/app
- Data models: src/model
- Controllers: src/controller
- Utility: src/utils

Naming convention:
- directory: kebab-case
- file: camelCase
- component: PascalCase
- asset: snake_case
